# API Overview

The public API is organized around four building blocks:

- `ztxexp.manager.ExpManager`
- `ztxexp.runner.ExpRunner`
- `ztxexp.analyzer.ResultAnalyzer`
- `ztxexp.pipeline.ExperimentPipeline`

Public dataclasses:

- `ztxexp.types.RunContext`
- `ztxexp.types.RunSummary`

All runtime artifacts in v0.2 follow schema version 2.
